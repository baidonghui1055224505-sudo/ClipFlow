功能特性

9 大核心能力，覆盖剪辑全流程

从素材获取到最终导出，无需切换工具。ClipFlow 用 AI 把复杂操作变成自然对话。

🎤

多语言语音转写

faster-whisper 引擎，99 种语言。VAD 静音过滤自动跳过空白。tiny 模式极速（1分钟/3秒），medium 模式高精度。

👥

说话人分离 + 改名

pyannote.audio 3.1 自动区分每个说话人。彩色标签标注，双击即可改名——「主持人」「嘉宾A」「受访者」。

🧠

AI 多轮对话剪辑

DeepSeek 大模型语义理解。说「保留讲产品优势的部分」「去掉寒喧」AI 自动匹配段落。不满意就补充一句，像聊天一样微调。

🔎

关键词搜索 + 跳转

输入关键词实时高亮匹配段落。Enter 跳转视频对应时刻，↑↓ 快速导航。快速定位关键发言。

✏

字幕原地编辑

转写有错别字？双击文字直接修改。修改后 SRT 导出和烧录字幕自动同步，无需二次操作。

🖨

三种导出方式

拼接 MP4（流拷贝秒出）· 烧录字幕（28px 白字黑边）· SRT 文件（导入 Pr / FCPX / 达芬奇）。

🌐

在线视频下载

粘贴 B站 / YouTube / TikTok / Vimeo 链接，自动下载。支持直链 URL。素材获取零门槛。

💻

跨平台 + 开机自启

macOS & Windows 原生支持。launchd 开机自动启动，崩溃自动恢复。就像系统自带工具一样可靠。

🔒

隐私优先，离线可用

转写和说话人分离模型全部本地运行。视频文件不会离开你的电脑。唯一的网络请求是向 AI 发送文字稿摘要。

📌 典型用法：周会录屏 → 说话人分离区分每位同事 → 搜索「 blocker 」找到关键讨论 → SRT 字幕存档，会后 5 分钟出纪要

📌 典型用法：录制 45 分钟 Python 教程 → 搜索「装饰器」直接跳到相关段落 → AI 筛选所有代码讲解段落 → 导出 15 分钟精华版 + SRT 字幕

📌 典型用法：5 场用户访谈视频 → 每场自动分离主持人和受访者 → 搜索「难用」汇集所有负面反馈 → SRT 分享给产品团队标注

📌 典型用法：90 分钟双人播客 → 说话人分离标注主持人和嘉宾 → AI 自动剪掉沉默和口误 → 导出拼接版 + SRT 字幕上传平台

📌 典型用法：B站直播回放 → AI 识别高能时刻 → 导出 5 个 60 秒精彩切片 → 烧录字幕直接发抖音/小红书
工作流

只需要四步，全程浏览器完成

没有时间轴，没有轨道拖拽，像聊天一样剪视频。

Step 1 上传视频

拖拽本地视频文件到浏览器，或粘贴 B站 / YouTube 链接自动下载。支持 mp4、mov、avi、mkv 等主流格式。

📤
🎤

Step 2 AI 转写 + 说话人识别

Whisper 模型将语音转成带时间戳的文字稿。可选开启说话人分离，自动识别不同发言者并用彩色标签标注。

Step 3 AI 对话剪辑

在对话面板用大白话告诉 AI 你要什么——「保留嘉宾聊技术的部分」「删掉开场废话」。不满意？再发一条补充，AI 会记住上下文。

🧠
🚀

Step 4 一键导出

拼接 MP4（流拷贝秒级完成）· 视频烧录字幕 · 标准 SRT 文件。任选一种，一键下载。

快速开始

一分钟跑起来

Python 3.9+ 和 FFmpeg 是仅有的两个依赖。其他一切自动处理。

1 安装 FFmpeg

# macOS
brew install ffmpeg
# Windows: ffmpeg.org 下载，解压到 C:\ffmpeg，加入 PATH

2 克隆 & 安装

git clone https://github.com/baidonghui1055224505-sudo/ClipFlow.git
cd ClipFlow
pip install -r backend/requirements.txt

3 配置 API Key（推荐）

# 创建 backend/.env:
DEEPSEEK_API_KEY=sk-your-key
HF_TOKEN=hf-your-token # 可选

DeepSeek 免费注册：platform.deepseek.com  |  约 ¥1/百万字
不配 Key 也能用：上传 → 转写 → 手动点选 → 导出

4 启动服务

# macOS
./start.sh
# Windows: 双击 start.bat

浏览器打开 http://localhost:8000
