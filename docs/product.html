功能特性

一切你需要的，都在这里

从转写到导出，ClipFlow 覆盖视频剪辑的完整流程。无需 Premiere，无需学习。

🎤

多语言语音转写

基于 faster-whisper，支持中英日韩等 99 种语言。VAD 静音过滤，1 分钟视频约 3 秒完成。tiny 模式极速，medium 模式高精度。

👥

说话人分离

pyannote.audio speaker-diarization-3.1 自动区分不同说话人。彩色标签标注，双击即可改名。播客、访谈、会议的福音。

🧠

AI 多轮对话剪辑

DeepSeek 大模型语义理解。用中文大白话描述需求——「保留讲产品优势的部分」「去掉开场寒喧」——AI 自动匹配段落。

🔎

关键词搜索 + 视频跳转

输入关键词实时高亮匹配段落，回车跳转视频对应时刻。上下箭头快速导航匹配项。

✏

字幕文字编辑

转写有错别字？双击任意段落文字即可原地修改。修改后 SRT 导出和烧录字幕同步更新。

🖨

三种导出方式

拼接 MP4 · 视频烧录字幕 · SRT 字幕文件。SRT 可导入 Premiere、Final Cut Pro、DaVinci Resolve。

🌐

在线视频下载

粘贴 B站、YouTube、Vimeo、TikTok 等平台链接，自动下载视频。也支持直链 URL 下载。

⚡

开机自启

macOS launchd 后台服务，开机自动启动，崩溃自动恢复。本地运行，数据不上传任何云端。

🖥️

跨平台支持

macOS 和 Windows 均可运行。Windows 用户双击 start.bat 即可启动，自动检测 FFmpeg 路径。

快速开始

一分钟跑起来

Python 3.9+ 和 FFmpeg 是唯一的依赖。

1. 安装 FFmpeg

# macOS
brew install ffmpeg

# Windows: 从 ffmpeg.org 下载，解压到 C:\ffmpeg，将 bin 目录加入 PATH

2. 克隆项目 & 安装依赖

git clone https://github.com/baidonghui1055224505-sudo/ClipFlow.git
cd ClipFlow
pip install -r backend/requirements.txt

3. 配置 API Key（可选但推荐）

# 创建 backend/.env，写入：
DEEPSEEK_API_KEY=sk-your-key
HF_TOKEN=hf-your-token # 可选，说话人分离需要

DeepSeek API Key 免费获取：platform.deepseek.com/api_keys  |  约 ¥1/百万字，充值 10 元够用很久
不配 API Key 也可用：上传 → 转写 → 手动点选段落 → 导出。只是没有 AI 帮你自动选。

4. 启动

# macOS
./start.sh

# Windows: 双击 start.bat

浏览器打开 http://localhost:8000，开始使用。

技术栈

完全本地，隐私优先

所有 AI 模型运行在本地。你的视频和音频数据不会上传到任何云端服务器——唯一的例外是 AI 对话功能会将文字稿摘要发送给 DeepSeek API 做语义匹配。

浏览器 (单页 HTML) ↓ HTTP / REST API FastAPI + Uvicorn (异步 Web 服务) ↓ 调用 faster-whisper 语音转写 | CPU, int8 量化 pyannote.audio 说话人分离 | speaker-diarization-3.1 DeepSeek Chat 语义理解 | 多轮对话 FFmpeg 视频处理 | 剪辑、拼接、字幕烧录 ↓ 持久化 本地文件系统 (临时文件 + 会话内存)
