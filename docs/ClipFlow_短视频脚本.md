# ClipFlow 短视频口播脚本

## 基本信息

- 时长：75 秒
- 画幅：竖屏 9:16（1080×1920）
- 风格：直给干货，少废话，语速偏快
- BGM：轻科技感，压低不盖人声

---

## 时间轴 + 画面 + 口播

| 时间 | 画面 | 口播 |
|------|------|------|
| 0-3s | 黑底白字弹出：**"剪辑一个 30 分钟的视频要多久？"** → 切到 ClipFlow 主界面全屏 | （等字幕出 1 秒后开口）剪辑一个 30 分钟的视频要多久？我用了 AI 之后，只需要一句话。 |
| 3-8s | 传统剪辑软件时间轴的混乱界面（网图/GIF），大红叉 ❌ 覆盖 | 以前你得学 PR、学剪映，拖时间轴拖到手抽筋。我干了这么多年设计，真受不了。 |
| 8-13s | 切回 ClipFlow，鼠标移到上传按钮，点击，文件快速上传 | 于是我自己写了一个工具。上传视频——它自动把文字稿给你转出来。 |
| 13-18s | 点击转写，2x 加速播放文字稿逐行出现，右侧文字稿区刷刷往下滚 | 还能识别视频里有几个人在说话——自动标出来，双击就能改名。 |
| 18-20s | 鼠标双击一个 speaker 标签，弹出改名框 | 主持人、嘉宾 A，一目了然。 |
| 20-30s | **重点画面**：鼠标移到 AI 对话框，慢动作敲字「保留讲产品优势的部分」，回车 | 最关键的一步来了——你不需要学任何剪辑操作。在对话框里说人话就行。比如——保留讲产品优势的部分。 |
| 30-36s | 右侧文字稿若干段落自动变蓝高亮（**这个是 WOW moment，放慢 0.5 倍**） | （停顿 0.5 秒）看到了吗？它自动帮我把跟产品优势相关的段落全挑出来了。不是关键词匹配，是 AI 真的理解了语义。 |
| 36-42s | AI 回复出现在对话框，鼠标划过蓝高亮的段落 | 而且你可以继续跟它对话——不对，把价格相关的去掉——它就在刚才的基础上再筛选。多轮对话，越来越精准。 |
| 42-50s | 点击导出按钮 → 下载进度条 → 下载完成提示 | 选好之后点导出，几十秒拼接完。还可以选——纯拼接的、带字幕烧录的、或者导出 SRT 字幕文件给 PR 用。 |
| 50-55s | 大字幕覆盖：**「本地免费 + AI 一次几分钱」** + DeepSeek 充值页面截图 | 听到这你可能想问，要不要花钱？语音识别是本地跑的，完全免费。AI 对话用的是 DeepSeek，一次几分钱，充 10 块钱够你用几个月。 |
| 55-65s | GitHub 仓库截图 → 鼠标划过 README → 指向 Star 按钮 | 代码我全部开源放在 GitHub 上了，搜 ClipFlow 就能找到。觉得有用的话随手点个 Star。 |
| 65-72s | 大字幕：**「让剪辑变成对话」** + ClipFlow 品牌名 | 以前剪视频是体力活，以后跟 AI 说句话就行。我是百东辉，关注我，后面还会分享更多 AI 工具。 |
