# ClipFlow 短视频内容策划（张咋啦 Zara 风格）

> 参考 Zara 的内容方法论：**活人感 + 动手实干 + 文科生友好 + 把复杂东西说人话**

---

## 一、内容策略总览

### 账号定位一句话
> 「让每个普通人都能用 AI 剪视频，不用学 Premiere，不用看教程，说人话就行。」

### 发布节奏建议
| 阶段 | 内容 | 数量 | 目标 |
|------|------|------|------|
| 预热期 | 痛点共鸣型 | 2-3 条 | 让人知道"原来这个问题可以这样解决" |
| 爆发期 | 功能演示型 | 3-4 条 | 展示核心功能，制造"哇塞" |
| 转化期 | 教程/对比型 | 2-3 条 | 引导下载使用 |
| 长尾期 | 用户案例/UGC | 持续 | 真实用户反馈 |

### Zara 风格核心公式
```
痛点暴击（3秒） → 反常识/情绪钩子（5秒） → 实操演示（40秒） → 核心卖点提炼（10秒） → 行动号召（5秒）
```

### 视频 1：痛点暴击 — 「你剪视频的样子真让人心疼」

**时长**：~60 秒  
**类型**：痛点共鸣  
**目的**：让目标用户产生"这就是我"的强烈共鸣，建立账号定位

---

| 时间 | 画面 | 口播（Zara 风格） |
|------|------|------|
| 0-3s | **封面字**：「你剪过视频吗？」黑底白字，突然切到一个人抓狂的表情 | （快语速）你剪过视频吗？不是那种加了滤镜发朋友圈的，是真的要把一个小时的视频剪成三分钟——那种。 |
| 3-8s | 屏幕录制：Premiere 复杂界面，时间轴密密麻麻 | 我一个朋友上礼拜跟我吐槽，说他录了个产品测评，20 分钟，想剪到 3 分钟，结果你猜怎么着——他在 Premiere 里泡了四个小时。四！个！小！时！ |
| 8-15s | 切换到 Zara 本人出镜，表情真诚 | 我当时就问他：你为什么不用 AI？他说试过，不好用。我说你用的什么？他说某个工具。我说——你用的那个不是 AI 剪辑，那叫语音转文字。 |
| 15-25s | 屏幕录制：打开 ClipFlow 界面，干净简洁 | 真正的 AI 剪辑是这样的——你上传视频，它自动给你转好文字稿。然后你不是去拖时间轴，你是——打字。对，就跟聊天一样。 |
| 25-35s | 演示：在对话框输入「把讲产品优势的部分挑出来」 | 比如说"把讲产品优势的部分挑出来"，回车。AI 自动帮你圈好段落。不满意？再来一句"嘉宾说的那部分也加上"——它记住了，它知道你在说什么。 |
| 35-45s | 演示：点击导出，选择带字幕 | 然后你点一下导出。拼接视频、带字幕的、纯字幕文件——三种格式随便选。三分钟。从上传到出片，三分钟。 |
| 45-50s | Zara 出镜 | 四个小时变三分钟。不是因为你变快了，是因为你用了对的工具。 |
| 50-55s | 黑底白字：「ClipFlow — 把剪辑变成一句话的事」 | 这个工具叫 ClipFlow，开源的，免费的，GitHub 上能下。 |
| 55-60s | Zara 出镜，微笑着说 | 链接我放主页了。别再说你不会剪视频了，现在你只是——还没打开这个网页。 |

### 视频 2：反常识 — 「AI 剪辑最厉害的不是 AI」

**时长**：~60 秒  
**类型**：观点/反常识  
**目的**：制造讨论，吸引泛科技人群

---

| 时间 | 画面 | 口播 |
|------|------|------|
| 0-5s | Zara 出镜，手里拿着咖啡 | （平静但有力）我跟你们说一个反常识的观点——AI 剪辑最厉害的地方，不是 AI。 |
| 5-10s | 快速闪过各种 AI 工具 logo | 不是它能帮你自动选段落，不是它转写有多快，不是它说话人分离有多准。 |
| 10-18s | 屏幕录制：ClipFlow 界面，光标在对话框里打字 | 是——**它让你用说话的方式剪视频**。你想一下，以前剪辑是什么？找时间轴、打标记、拖轨道、切刀片工具。每一个操作都在问你：你是个专业的剪辑师吗？ |
| 18-28s | 屏幕录制：ClipFlow 对话框，用户输入「保留嘉宾聊技术的部分，去掉开场废话」 | 但 ClipFlow 的逻辑是——你不需要懂剪辑。你只需要会说话。"保留嘉宾聊技术那部分""删掉开头的废话""只要讲干货的"。这不就是你跟同事交代任务的语气吗？ |
| 28-38s | Zara 出镜 | 这件事最妙的地方在于——一旦你把剪辑变成了对话，你就可以反复微调。AI 选的不是你想要的？多说一句就行。它记得住上下文。这叫什么？这叫**多轮对话**。这在其他剪辑工具里是不存在的。 |
| 38-48s | 屏幕录制：演示多轮对话，第一次 AI 选了 5 段，第二轮补充后选了 8 段 | 第一轮它选了 5 段。你说"把刚才漏掉的嘉宾发言也加上"——它又找出来 3 段。你不需要从头来一遍，它知道你在说什么。 |
| 48-55s | Zara 出镜 | 所以你看，AI 最厉害的不是替代人。是让一个完全不会剪辑的人，能用他自己最舒服的方式——说话——来完成剪辑。这才是 AI 的意义。 |
| 55-60s | 字幕：「ClipFlow  GitHub 开源」| 工具叫 ClipFlow，开源免费。去试试，你都不需要会看时间轴。 |

### 视频 3：功能炸裂 — 「这个工具把我整不会了」

**时长**：~75 秒  
**类型**：功能演示合集  
**目的**：密集展示功能，制造"这也行？那也行？"的冲击感

---

| 时间 | 画面 | 口播 |
|------|------|------|
| 0-4s | Zara 出镜，表情震惊 | （瞪大眼睛）我本来只是想找个工具帮我剪播客。结果这个工具把我整不会了——它功能也太多了。 |
| 4-12s | 快速闪切：6 个功能画面各 1.5 秒 | 来，你感受一下：语音转写 99 种语言、AI 对话剪辑、说话人分离、关键词搜索直接跳视频、双击改字幕、在线下载 B站 YouTube 视频——我：？？？这是一个工具？ |
| 12-20s | 屏幕录制：上传一个 YouTube 链接 | 你甚至不用下载视频——把 B 站或者 YouTube 链接直接粘贴进来，它自动把视频拉下来。对，你连"找素材"这一步都省了。 |
| 20-30s | 屏幕录制：转写完成，显示说话人标签 | 然后它自动转写。注意看——如果视频里有不同的人说话，它会给每个人标不同颜色。主持人、嘉宾 A、嘉宾 B。你双击标签就能改名。播客博主的刚需，认不认？ |
| 30-40s | 屏幕录制：搜索框输入关键词，高亮匹配 | 文字稿出来以后，你可以直接搜关键词。比如搜"钱"——所有提到钱的段落全给你标出来。点一下视频就跳过去。谁说视频不能 Ctrl+F 的？现在可以了。 |
| 40-48s | 屏幕录制：双击文字稿，改成正确文本 | 还有一个细节我特别喜欢——它转写偶尔会有错别字。你双击那个字，直接改。改完以后导出 SRT 字幕，也是改完的版本。不用改两遍。 |
| 48-58s | 屏幕录制：导出选项，三个按钮 | 导出的时候你有三个选项——普通拼接、带上字幕的视频、还有 SRT 字幕文件。SRT 的意思是你扔进 Premiere、Final Cut、达芬奇都能用。 |
| 58-68s | Zara 出镜 | 我之前一直说，好工具的标准不是功能多，是让你**不用动脑子**。ClipFlow 就是你脑子里想"我要把这段剪掉"——它帮你做了。你甚至不需要知道"剪辑"两个字怎么写。 |
| 68-75s | 字幕：「ClipFlow 开源免费  GitHub 可下」| 开源免费，GitHub 上能下。链接在主页。去试试，你会发现剪辑真的可以不用学。 |

## 五、补充：如果只拍一条

如果时间精力有限，**优先拍视频 #4「场景代入」**。

原因：
1. 6 种人自动对号入座 → 覆盖面最广
2. 不需要深度演示 → 制作成本最低
3. 评论区互动最强 → "我是XX，这个太需要了"
4. 最容易引发转发 → "@你那个做播客的朋友"

拍完 #4 看数据，哪个场景讨论最多，再针对那个场景拍一条深度的。
